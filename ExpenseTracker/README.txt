# Expense Tracker

A simple Java-based application for tracking daily expenses. The app allows users to add new expenses, view all expenses, generate a report of the recorded expenses, and reset the expense tracker.

## Features

- **Add Expense:** Allows users to input an amount and a category for each expense.
- **Generate Report:** Summarizes all recorded expenses, displaying their categories, amounts, and the total amount spent.
- **Show All Expenses:** Displays a list of all expenses recorded.
- **Reset Tracker:** Clears all stored expenses and resets the tracker.
  
## Technologies Used

- Java
- Swing (for GUI)
- Java Event Handling

## Requirements

- Java 8 or higher
- IDE that supports Java (e.g., IntelliJ IDEA, Eclipse, NetBeans)

Open the project in your preferred Java IDE.

Run the ExpenseTracker class to start the application.

Usage
Add an Expense:
Enter the amount and category for the expense in the input fields, and click "Add Expense". The expense will be recorded.

Generate Report:
Click the "Generate Report" button to view a summary of your recorded expenses, including the total amount spent.

Show All Expenses:
Click "Show All Expenses" to view all the expenses entered into the tracker.

Reset Tracker:
Click "Reset Tracker" to clear all stored expenses and reset the application.
