# Simple Notepad

A simple Java-based Notepad application that provides basic text editing features like creating a new file, opening existing files, saving files, undoing actions, and searching for text.

## Features

- **New File:** Create a new empty document.
- **Open File:** Open an existing text file.
- **Save File:** Save the current document.
- **Save As:** Save the current document with a new name.
- **Undo:** Undo the last action.
- **Edit Operations:** Cut, Copy, and Paste text.
- **Find Text:** Search for specific text within the document.
- **Error Handling:** Displays error messages for file-related issues.

## Technologies Used

- Java
- Swing (for GUI)
- Java I/O (for file operations)
- UndoManager (for undo functionality)

## Requirements

- Java 8 or higher
- IDE that supports Java (e.g., IntelliJ IDEA, Eclipse, NetBeans)


Usage
New File: Click "File" -> "New" to start a new file.
Open File: Click "File" -> "Open..." to open an existing file from your computer.
Save File: Click "File" -> "Save" to save the current file.
Save As: Click "File" -> "Save As..." to save the file with a new name.
Undo: Click "Edit" -> "Undo" to undo the last action.
Find Text: Click "Edit" -> "Find..." to search for specific text within the document.
